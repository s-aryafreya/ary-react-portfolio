 🪐 Saturnian Moons Portfolio
**Build v0.9.5 (Stable)** | Developed by Salem Arya Freya

A 1990s desktop-inspired portfolio built with **React Native Web**. This project serves as a central hub for my mobile development projects, interactive media, and leadership experience at UCF.

 🚀 Key Features
* **Retro UI Engine:** Custom-built components mimicking the Windows 95/98 aesthetic using `StyleSheet` beveling.
* **Live Systems:** A real-time system clock and dynamic weather-safe header.
* **Interactive Terminal:** A functional contact form with logic-based validation and "System Error" messaging.
* **Animated Transitions:** Smooth window entry effects using the React Native `Animated` API.

 🛠️ Tech Stack
* **Frontend:** React Native (Web), Expo
* **Styling:** JavaScript StyleSheets with 98.css influence
* **Deployment:** GitHub Pages
* **Typography:** Custom W95FA OpenType font

 📂 Project Structure
* `/components`: Modular UI elements (`Header.js`, `RetroWindow.js`)
* `/assets`: Local storage for custom fonts and project screenshots
* `App.js`: Main application logic and state management
